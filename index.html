GET

https://apipostrejs.fly.dev/[ nombre de la base de datos ]

Se usa el nombre de la tabla que se desee consultar

https://jspastelapi.fly.dev/postre
POST

https://jspastelapi.fly.dev/insertar?id=[ ]&precio=[ ]&sabor=[ ]&tamano=[m]

Se usa el id, precio, sabor y tamaño de la tabla que se desee agregar

https://jspastelapi.fly.dev/insertar?id=1&precio=100&sabor=fresa&tamano=mediano
PUT

https://jspastelapi.fly.dev/actualizar?id=[ ]&precio=[ ]&sabor=[ ]&tamano=[ ]

Se usa el id y sección de la tabla que se desee actualizar

https://jspastelapi.fly.dev/actualizar?id=1&precio=100&sabor=fresa&tamano=grande
DELETE

https://jspastelapi.fly.dev/eliminar?id=[ ]
